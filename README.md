
# Prueba GazeLook



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```


## Api Documentation

``` 
$ http://localhost:5000/api/documentation/
```
