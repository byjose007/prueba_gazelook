
# Prueba GazeLook



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```

## Tecnologías:
- TypeScript
- MongoDB.
- Node JS (Nestjs Framework)
- Google Cloud Storage

## Arquitectura
- Clean Architecture
